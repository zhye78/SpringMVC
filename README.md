# SpringMVC
Spring MVC + Data Binding + Data Validation 을 사용하여 Offer 관리 폼 만들기

### 구현 기능
1. 로그인
2. 로그아웃
3. 전체 Offers 조회
4. 새 Offer 추가
5. Offer 추가시 validateion 기능 사용하여 유효한 값만 입력 가능하도록 함
